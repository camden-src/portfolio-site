# portolfio-site
Front end code for an artist's portfolio site.

This is of little potential value except to the author in building his personal creative web presense by leveraging tools he has become familiar with professionally, but has developed no specialization.

As such the author knows enough JavaScript, HTML, and CSS to make a convincing website after some iteration. The absense of a modern web framework is intentional, as the author is only familiar with rather antiquated frameworks and wishes to use the time involved in learning an updated framework to address other engineering concerns around site functionality and design.

At time of writing this is most certainly not a convincing website, but it is a workable proof of concept upon which to build a convining one.
